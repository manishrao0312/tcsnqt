# 🎯 TCS NQT Coding Mastery List

A self-hosted interactive coding checklist for **TCS NQT 2026 Digital & Prime tracks** — 136 hand-picked questions with full topic coverage, progress tracking, and direct links to LeetCode/GFG.

🔗 **Live:** [tcsnqt-green.vercel.app](https://tcsnqt-green.vercel.app)

---

## ✨ Features

- **136 curated problems** covering all TCS NQT topics
- 🔴 **Must-Do** tags for questions that appear in almost every TCS slot
- ✅ **Click to mark done** — progress saves locally in the browser
- 🏷️ Filter by status: Pending / Done / Must-Do / Easy / Medium / Hard / Newly Added
- 📊 Live progress counter (Done / Left / % complete)
- Direct links to **LeetCode** and **GFG** for every problem


---

## 💡 How to Use

1. Start with 🔴 **Must-Do** problems — these appear in almost every TCS slot
2. Solve in order — topics sorted by TCS frequency
3. Click any row to mark it done; progress is saved in `localStorage`
4. Time targets: 30 min Easy · 45 min Medium · 60 min Hard
5. Always handle edge cases: empty input, single element, all-same values
